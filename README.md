# IronLog Workout Tracker

## Project Description
IronLog is an Android mobile application designed to help users track workouts, exercises, and fitness progress. The app allows users to log training sessions, record sets, reps, and weight, and review previous workouts directly from their Android device.

## Problem Being Addressed
Many workout tracking apps are overly complicated, filled with unnecessary features, or require paid subscriptions. IronLog provides a simple and organized solution for users who want to track their workouts and monitor progress without distractions.

## Platform
Android mobile application.

## Front-End Support
- Kotlin
- XML layouts or Jetpack Compose
- Android Studio
- Material Design components

## Back-End Support
- Room Database for local storage
- Optional Firebase support for user authentication and cloud syncing
- WorkManager for background syncing or reminder tasks

## Functionality
- User login and registration
- Create workouts
- Add exercises
- Track sets, reps, and weight
- View workout history
- Track progress over time
- Optional workout reminders

## Design (Wireframes)
- Login screen
- Home dashboard
- Workout creation screen
- Exercise logging screen
- Workout history screen
- Progress tracking screen
